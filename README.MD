## 🚀 Features

### Interactive Elements
- **Custom Cursor**: Unique cursor design with dot and ring following mouse movement
- **Matrix-Style Background**: Animated canvas with falling code characters, floating particles, and network nodes
- **Smooth Animations**: Fade-in effects and scroll-based reveals using Intersection Observer API
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices

### Sections
1. **Hero Section**: Dynamic introduction with professional stats and watermark text
2. **About**: Personal introduction and detailed information grid
3. **Experience**: Timeline of professional roles with responsibilities
4. **Skills**: Interactive skill bars with category tabs (Testing, Tools, Tech, Soft Skills)
5. **Projects**: Grid showcase of major projects with descriptions
6. **Certifications**: Display of professional certifications
7. **Awards**: Recognition and achievements
8. **Contact**: Easy-to-use contact information with multiple channels

### Technical Highlights
- **Zero Dependencies**: Pure HTML, CSS, and vanilla JavaScript
- **Self-Contained**: Single-file HTML portfolio (41KB)
- **Custom Animations**: Canvas-based particle system with QA-themed symbols
- **Performance Optimized**: Smooth 60fps animations
- **Accessibility**: Semantic HTML and keyboard navigation support

## 🛠️ Technologies Used

## 🚀 Getting Started

### Installation

1. Clone or download the repository
```bash
git clone https://github.com/Nagabharana/portfolio.git
```

2. Open the file
```bash
cd portfolio
```

3. Launch in browser
- Simply open `index.html` in any modern web browser
- Or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

4. View in browser
```

```

## 🔧 Customization

### Update Content
All content is embedded in the HTML file. To customize:

1. **Personal Information**: Update the hero section text and stats
2. **Experience**: Modify the experience section with your roles
3. **Skills**: Edit the `SD` object in the JavaScript section
4. **Projects**: Update project cards with your work
5. **Contact**: Change contact links and information

### Styling
CSS custom properties are defined in `:root`:
```css
--bg: #0a0a0a;      /* Background */
--acc: #e8ff00;     /* Accent color */
--blu: #4a9eff;     /* Blue accent */
--txt: #f0f0f0;     /* Text color */
```

### Canvas Animation
Customize animation parameters in the JavaScript:
- `MC`: Matrix characters array
- `SY`: Floating symbols array
- Particle count, speed, and behavior
