# QA Engineer Portfolio

A modern, interactive portfolio website showcasing QA engineering expertise with animated tech-inspired visuals.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- Custom animated cursor with dot and ring effects
- Matrix-style background with falling code and floating particles
- Interactive skill bars with category filtering
- Smooth scroll animations and transitions
- Fully responsive design for all devices
- Single-file implementation with zero dependencies

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (Canvas API, Intersection Observer)
- Google Fonts (Space Mono)

## 📋 Sections

- **Hero** - Introduction with professional stats
- **About** - Background and information grid
- **Experience** - Professional timeline
- **Skills** - Interactive skill categories
- **Projects** - Portfolio showcase
- **Certifications** - Professional credentials
- **Awards** - Recognition and achievements
- **Contact** - Multiple contact channels

## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/Nagabharana/portfolio.git
```

2. Open `index.html` in your browser

That's it! No build process or dependencies required.

## 📱 Responsive

- Desktop: Full animations with custom cursor
- Tablet: Touch-optimized navigation
- Mobile: Hamburger menu and stacked layouts
